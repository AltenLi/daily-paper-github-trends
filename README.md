# 📚 每日论文 & GitHub Trending 日报

自动汇总每日 AI/CS 领域 arXiv 新论文与 GitHub 热门仓库，附中文简介，帮开发者/研究生节省筛选时间。

## 数据来源

本项目计划自动追踪并整理以下公开数据源：

- **arXiv API**：重点关注 `cs.AI`、`cs.CV`、`cs.CL`、`cs.LG`、`cs.IR` 等 AI/CS 相关方向。
- **GitHub Trending API**：追踪每日热门开源项目，尤其关注 AI 工程化、RAG、Agent、搜索、LLM 应用、开发者工具等方向。

## 每日内容

后续每日归档会包含：

- arXiv 最新论文列表
- GitHub Trending 热门项目列表
- 中文简介 / 摘要
- 原始链接
- 主题标签
- 适合开发者和研究生快速浏览的筛选信息

完整版每日推送放置于微信公众号「AltenAI观察」，欢迎搜索关注获取。

## 目录结构

当前目录结构会随着脚本和数据归档逐步补充，计划如下：

```text
.
├── README.md              # 项目说明
├── LICENSE                # MIT License
├── .gitignore             # Python 项目忽略规则
├── scripts/               # 后续放置抓取、筛选、生成日报脚本
├── data/                  # 后续放置每日原始/结构化数据
└── daily/                 # 后续放置每日 Markdown 日报归档
```
